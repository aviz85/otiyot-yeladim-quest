# אותיות וילדים: תעלומת הכתבה שנעלמה 🗞️

משחק הרפתקאות פיקסל-ארט בעברית לילדים, בסגנון משחקי ההצבע-ולחץ של שנות ה-90 (בהשראת LucasArts).

נעם, קורא צעיר שזכה ביום במערכת העיתון, מגלה שהכתבה הראשית של הגיליון החגיגי נעלמה — שלושה עמודים התפזרו לכל עבר, והדדליין בשש בערב. חתול חשוד, יונה סוחטת גרעינים, ארכיון חשוך וחידות משעשעות בין לבין.

## ⚠️ גילוי נאות

**כל העלילה, הדמויות והאירועים במשחק דמיוניים לחלוטין** ואין ביניהם לבין אנשים אמיתיים או המערכת האמיתית של עיתון ״אותיות וילדים״ שום קשר. **המשחק כולו — סיפור, גרפיקה, מוזיקה ודיאלוגים — נוצר על ידי בינה מלאכותית**, באהבה ובלי שום כוונה לפגוע.

## 🎮 איך משחקים

- **עכבר:** לוחצים על דברים. ריחוף מציג את שם האובייקט. קליק ימני (או כפתור 👁️) = להסתכל.
- **מקלדת:** חצים לתנועה, Enter לאינטראקציה, רווח להמשך דיאלוג, 1-9 לבחירת תשובה.
- **💡 רמז** — אם נתקעתם. **💾 שמירה / 📂 טעינה** — נשמר בדפדפן (localStorage), כולל שמירה אוטומטית.
- מוזיקת צ׳יפטיון 16-ביט מקורית בכל חדר (🎵 להשתקה).

## 🛠️ טכנולוגיה

Vanilla JS + Canvas — בלי תלויות, בלי build. כל הפיקסל-ארט מצויר בקוד, כל המוזיקה מסונתזת ב-Web Audio.

```bash
# להרצה מקומית
python3 -m http.server 8080
# ואז http://localhost:8080
```
